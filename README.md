# cre-dashboard

Requirements:

* npm & node
* tested with 
  * npm 6.4.1
  * node v11.1.0

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production 
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Deployment

Firebase cli tool is required
```
npm install -g firebase-tools
```

```
firebase login // opens web browser

firebase deploy
```
